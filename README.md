# DVrepo
##### how to run DVproj
###### clone
```
git clone https://github.com/krys613/DVproj.git
```
###### need
node.js
###### get the patent data
1. find DVdatabase
2. import two sql file in mysql
3. change the configuration in DVserver->mysql->pool.js and connect to your mysql
###### run server
1. cd DVserver 
2. npm install
3. npm start
###### run website
1. cd DVweb
2. npm install
3. npm start
4. visit http://localhost:8080
